# Unity Slot Machine

A slot machine game developed in Unity featuring randomized reel spinning, betting, coin management, lever animation, and sound effects.

## Game Overview

Players place a bet, spin the slot machine, and win rewards by matching all three middle symbols.

## Running the Web Build

The Web build is available in the `Build/WebGL` folder. Run it using a local web server or deploy it to a web hosting service.

## Bonus Features

- Betting system
- Coin management
- Sequential reel stopping
- Lever animation
- Sound effects
- Responsive UI

## Approach

The project was developed using a modular approach by separating reel control, game management, betting, and animation into independent scripts for better organization and maintainability.
